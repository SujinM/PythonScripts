# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-22

### Added
- Initial release
- Admin configuration module for user enrollment and management
- Fingerprint reader module for authentication
- GPIO integration for hardware control
- Configuration file management
- Logging system
- Command-line interfaces for admin and reader modes
- Comprehensive documentation
- Unit tests
- Examples

### Features
- Enroll new fingerprints
- Delete enrolled users
- Verify fingerprints
- Store fingerprint images
- User configuration management
- GPIO trigger on successful authentication
