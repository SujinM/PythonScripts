# Place application icon files here.
#
# Expected files:
#   app.ico       – 256×256 multi-resolution Windows icon (used by setup.py for .exe icon)
#   app.png       – 256×256 PNG fallback for non-Windows platforms
#   connect.png   – 32×32 toolbar icon for the Connect action (optional)
#   disconnect.png– 32×32 toolbar icon for the Disconnect action (optional)
#
# Icons are loaded at runtime from this directory via ICONS_DIR (constants.py).
# The .ico file is referenced in setup.py for the Windows executable.
#
# If app.ico doesn't exist, cx_Freeze will use a default Windows icon.
#
# Where to get icons:
#   - Free icons: https://icons8.com or https://www.flaticon.com
#   - Convert PNG to ICO: https://convertio.co/png-ico/
#   - Create custom: Use GIMP, Paint.NET, or online tools
#
# TIP: You can copy an .ico from another project as a placeholder.
# For example: Copy-Item "..\BackupScript\assets\backup_icon.ico" "app.ico"
